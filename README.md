# visual-statistic

インタラクティブな統計可視化ダッシュボード。外部依存なしの単一 HTML ファイルで動作します。

## デモ

ブラウザで HTML ファイルを開くだけで使えます。インストール不要。

```bash
open proportion-ci.html
```

## コンテンツ

### 割合の信頼区間 (`proportion-ci.html`)

二項比率の信頼区間を3手法で比較するビジュアライザー。

**機能**

- **3手法の比較**: Wilson 区間 / Clopper-Pearson（正確区間）/ 正規近似（Wald）を並べて表示
- **パラメータ入力**: サンプルサイズ *n* と成功回数 *k* をスライダーまたは数値入力で設定
- **信頼水準切り替え**: 90% / 95% / 99%
- **手法トグル**: 左パネルで手法を個別に表示/非表示。各手法の区間値もリアルタイム表示
- **Beta 事後分布グラフ**: Beta(*k*+1, *n*-*k*+1) の密度曲線に CI 区間をシェードで重ねて表示
- **プリセット**: 標準サンプル / 小サンプル / 稀少事象 / 大サンプル / 境界付近
- **警告表示**: 正規近似の適用条件（*np̂* < 5 等）を自動チェック
- **プロンプト出力**: 現在の設定を自然言語でまとめてコピーできる

**各手法の特徴**

| 手法 | 特徴 |
|------|------|
| Wilson | スコア検定ベース。小 *n* や極端な *p̂* でも安定（推奨） |
| Clopper-Pearson | ベータ分布の正確区間。常に保守的で名目水準を保証 |
| 正規近似 (Wald) | 中心極限定理による近似。大 *n* かつ *p̂* ≈ 0.5 でのみ適切 |

## 技術

- 単一 HTML ファイル（CSS・JS インライン）
- 外部ライブラリ・CDN 依存なし
- グラフ描画: SVG（手書き生成）
- 数値計算: Lanczos 近似 lgamma、連分数展開 betainc、Newton 法 betainv
